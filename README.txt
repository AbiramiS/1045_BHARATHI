7. Install (Installation procedure): 

How to run the project ?


1. Copy the Bharathi.apk from Demo folder to the SD card of the Android device and click it using a file navigator from the device.


2. In the Android Device goto Settings > Langauge and input, and check Bharathi.


3. To use Bharathi goto, any of the text feild in the android device, you will noice a small keyboard icon in the notification area, choose Bharathi.
